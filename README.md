# Komodo-Hub

## Brief

Hello everyone, please use this Gitea for managing the project.

Please commit only your own changes (do not get someone else to commit your changes for you), since we can use this as evidence
and help in the write-up to know exactly what code you've created (that's a requirement for the write-up).

## Where to start

### Issues

Go to the issue tab at the top of the repository page. You may find there are already issues assigned to you.
If not, find an issue that is labeled as `Help Wanted`. Issues labeled as `High priority` should all be done before
issues marked as `Medium priority` and of course `Low priority`.

If you find one that you want to start working on, please add a comment to let others know, so we can
avoid duplicate work and unnecessary merge conflicts.

I will assign the issue to you, ensuring your correctly credited for the work you've added.



If there's a bug in the code that you either haven't been assigned to or don't know how to fix, open an issue explaining the bug and how
to repeat it.

After finishing an issue (or partly finishing it), add a comment to the specific thing you were fixing, with the commit ID referenced.

### Git

Clone the git repo by using `git clone https://git.wbell.dev/Ugric/Komodo-Hub`. This will create a `Komodo-Hub` directory in the CWD.

Please give a relevant commit message and an issue ID if one is related (e.g. for an issue ID of 7 add `#7` to the start of your commit message).
